# VR 교육 플랫폼

인지가 느린 아이들을 위한 VR 시뮬레이션 교육 플랫폼입니다.

## 프로젝트 개요

이 프로젝트는 VR 기술을 활용하여 인지가 느린 아이들이 일상생활 기술을 안전하고 체계적으로 배울 수 있도록 돕는 교육 플랫폼입니다.

## 주요 기능

### 사용자 역할
- **부모님**: 아이를 등록하고 학습 기록을 확인할 수 있습니다.
- **교육자(선생님)**: 여러 아이들을 관리하고 학습 진행 상황을 모니터링할 수 있습니다.

### 교육 시나리오
1. **횡단보도 건너기**: 신호등을 확인하고 안전하게 횡단보도를 건너는 시나리오
2. **대중교통 이용하기**: 버스를 타고 목적지까지 안전하게 이동하는 시나리오
3. **키오스크로 주문하기**: 키오스크를 사용하여 음식을 주문하고 결제하는 시나리오
4. **무인 편의점 이용하기**: 편의점에서 물건을 구매하고 계산하는 시나리오

### 주요 페이지
- **메인 페이지**: 플랫폼 소개 및 시나리오 카드
- **로그인/회원가입**: 부모님 또는 교육자 계정 생성 및 로그인
- **마이페이지**: 아이 관리 및 학습 기록 조회
- **시나리오 선택**: 체험할 시나리오 선택
- **아이 선택**: 시나리오를 체험할 아이 선택 및 VR 코드 생성
- **시나리오 상세**: 점수, 단계별 완료 현황, AI 분석 결과 확인

## 기술 스택

- React 18.2.0
- React Router DOM 6.20.0
- Vite 5.0.8
- CSS3

## 설치 및 실행

### 필수 요구사항
- Node.js 16.x 이상
- npm 또는 yarn

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 `http://localhost:3000`으로 접속하세요.

### 빌드
```bash
npm run build
```

- `https://` - 백엔드 서버 주소
- 모든 요청/응답은 `snake_case` 형식을 사용합니다.

## 주요 기능 설명

### VR 코드 생성
시나리오와 아이를 선택하면 5자리 랜덤 코드가 생성됩니다. 이 코드를 VR 기기에서 입력하면 해당 시나리오가 시작됩니다.

### 학습 기록
- 각 시나리오별 점수 기록
- 단계별 완료 현황
- 실수한 부분 기록
- AI 기반 분석 결과 (잘한 점, 개선할 점, 추천사항)
